## 🎯 讲述技巧练习

### 技巧1: 语速控制

**练习方法**:
1. 录音讲述一遍
2. 听录音，检查语速是否适中
3. 调整语速，重新录音
4. 重复直到满意

**检查清单**:
- [ ] 不要说得太快 (让面试官能跟上)
- [ ] 不要说得太慢 (浪费时间)
- [ ] 在关键词处停顿 (让面试官反应)
- [ ] 在部分之间停顿 (2-3秒)

---

### 技巧2: 强调关键词

**练习方法**:
1. 标记出讲述中的关键词
2. 在讲述时强调这些关键词
3. 录音检查是否有效强调

**关键词列表**:
- 30-60分钟 (Ansible部署时间)
- 5-10分钟 (Jenkins部署时间)
- 8阶段 (Jenkins流程)
- 4层架构 (整体架构)
- 4个核心成就 (项目成就)
- 完全自动化 (关键特性)
- 高可用 (关键特性)

## 💡 常见问题

### Q: 讲述时忘记了怎么办？
```
A: 不要紧张，深呼吸。
   可以说："让我想一下..."
   然后继续讲述。
   不要停顿太久。
```

### Q: 讲述时说错了怎么办？
```
A: 不要紧张，继续讲述。
   可以说："我刚才说错了，应该是..."
   然后继续讲述。
   不要反复纠正。
```

### Q: 讲述时超时了怎么办？
```
A: 加快语速，但不要太快。
   删除不重要的细节。
   保留关键内容。
```

### Q: 讲述时面试官打断了怎么办？
```
A: 停止讲述，回答面试官的问题。
   回答完后，继续讲述。
   或者等面试官让你继续。
```
